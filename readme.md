# 🛡️ Anti-Fraud ETL System

Система автоматизированного сбора транзакционных данных и выявления подозрительных операций по картам клиентов.

## 📋 Описание логики
Проект реализует ETL-процесс (Extract, Transform, Load) из сырых CSV-файлов в хранилище данных (DWH) на базе PostgreSQL.

## **Основные проверки:**
1. **Черный список паспортов:** детекция операций клиентов с недействительными документами.
2. **Географическая аномалия:** Выявление транзакций по разным счетам одного клиента из разных городов в короткий промежуток времени.
3. **попытки подбора сумм:**  попытки снятия средств подбором - при 3-х неудачных, где каждая следующая на меньшую сумму, чем предыдущая 
4. **просроченный договор обслуживания:** Проведение операций при истекшем договоре обслуживания
